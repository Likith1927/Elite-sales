# EliteSales Solutions - Project Guidelines

## Aesthetic & Design
- **Minimalist & Clean**: Use a palette of whites, light grays, and a single professional accent color (e.g., Deep Navy #1A202C).
- **Professional & Trustworthy**: Focus on high-quality typography (Inter) and spacious layouts.
- **E-commerce Feel**: Use smooth transitions (motion), clear call-to-actions, and high-end service cards.

## Core Features
- **Application Portal**: A clean form collecting LinkedIn, Gmail, User Name, Contact Info, Company Name, and Goals.
- **AI Evaluation (MCP Tool)**: Use Gemini API to evaluate if an application matches company requirements.
- **Automated Response**: Simulate/Trigger automated emails based on AI evaluation (Selected/Rejected).

## Technical Stack
- **Frontend**: React 19, Tailwind CSS 4, Motion.
- **Backend**: Express (Full-stack Vite middleware).
- **AI**: Google GenAI SDK (@google/genai) for application processing.
